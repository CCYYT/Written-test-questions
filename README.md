# 拉取破价商品链接
方案：每次获取的链接都 转移到已获取的链接List中，使用锁保证并发调用接口的时候 不能返回同样的数据

# 上传截图
方案一：客户端上传截图失败后，调用添加链接接口，将链接归还；
方案二：每次返回了的链接都放在一个List中保存，当客户端上传完截图后，在到这个List里删除。并且开启定时任务，每隔一段时间就将这个List中的链接 归还回去。
